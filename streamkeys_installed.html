A new version of Streamkeys has been installed!

...This one works!

This is a major update that fixes the majority of the problems from the previous release. If this extension didn't work for you before ... try it again now! The hotkey presses are now captured more accurately and you will be able to control the players from anywhere in the Chrome browser. In addition, support for many new sites has been added.
If you still run into problems please send me an email: minilogo@gmail.com

New Default Hotkeys:

Mac:

Play/Pause: Alt + Shift + 2
Next Song: Alt + Shift + 3
Previous Song: Alt + Shift + 1
Mute: Cmd + Shift + M

Windows:

Play/Pause: Media Key Play
Next Song: Media Key Next
Previous Song: Media Key Previous
Mute: Cmd + Shift + M

Note: All hotkeys can be changed. Simply go to the chrome extensions tab (get there from Chrome menu -> tools -> extensions). At the bottom there is a button for "Keboard Shortcuts" where you can change the hotkeys to those of your choosing.

Mac users: Currently there is an issue with capturing media key presses in chrome on macs. This issue is slated to be fixed in the next Chrome release, but until then you will have to use other keys, sorry!

Thanks for using Streamkeys!
You can change these hotkeys to anything you like - on the chrome extension tab there is a link at the very bottom called "Keyboard shortcuts".
